# EcoDJ-Py
A music bot for discord servers, made with Python 3.10.2

To search for music he uses Wavelink and Lavalink

# Musics Commands:
▪︎ !connect
▪︎ !disconnect
▪︎ !play <query>
▪︎ !skip
▪︎ !pause
▪︎ !resume
▪︎ !seek <seconds> [reverse=False]
▪︎ !volume <vol> [forced=False]
▪︎ !loop <type>
▪︎ !nowplaying
▪︎ !queue
▪︎ !equalizer
▪︎ !lyrics [query]
